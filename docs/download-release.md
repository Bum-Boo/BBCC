## Release Notes To Keep Visible

- BBCC is Windows-first.
- Supported practical controller focus is 8BitDo Zero 2 and Xbox/XInput-family devices.
- The app can send local keyboard and mouse input to the foreground app, so safety/stop/reset guidance should stay visible.
- Release screenshots should use safe demo mappings and no private workflow data.
