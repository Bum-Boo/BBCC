# BBCC v0.1.0

BBCC v0.1.0 is the first public release of the project. It introduces the core controller-mapping workflow for creative apps, desktop navigation, and media control, with a Windows-first focus and an interface built around remembered devices and editable profiles.

## Included Features

- Tray-resident controller mapping workflow
- Device -> App Profile -> Preset -> Button Mapping model
- App-based profile switching
- Per-profile presets
- `YouTube` / media fallback profile
- Offline saved-device editing for remembered controllers
- Multilingual UI support
- Theme support
- Controller-aware layouts and diagrams
- Inspector and mapping editor for reviewing and adjusting bindings

## Supported Controllers

- 8BitDo Zero 2
- Xbox Controller / XInput-family devices
- Additional controller models can be requested through issues

## Notes

- BBCC is a Windows-first project
- Browser-based media fallback behavior is part of the broader media workflow
- Layouts, mappings, and controller handling are still being refined
- Some device handling is expected to evolve as controller coverage expands

If you want support for another controller model, please open an issue with the device name and a short description of the expected behavior.
